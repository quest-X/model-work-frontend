# 测试AI推理功能

## 测试步骤：

1. **启动应用**
   ```bash
   npm run dev
   ```

2. **加载图片**
   - 将图片拖拽到应用中，或点击上传

3. **选择矩形标注工具**
   - 确保选择了矩形标注模式

4. **绘制标注框**
   - 在图片上拖拽绘制一个矩形框
   - 松开鼠标完成标注

5. **观察反馈**
   - 查看页面顶部是否出现 "推理中" 通知
   - 等待推理完成通知

6. **查看结果**
   - 点击右侧 "推理结果" 按钮
   - 查看检测到的对象信息

## 调试检查：

### 浏览器控制台 (F12)
应该看到类似输出：
```
Rect label created, triggering segmentation for rect: {...}
Starting segmentation for bbox: {...}
Calling segmentation API with bbox: x1,y1,x2,y2
```

### 网络请求 (F12 → Network)
- 寻找对 `/segment` 的POST请求
- 检查请求状态码和响应数据

### 推理服务器状态
确保推理服务器正在运行：
- 地址: http://192.168.10.205:8000/segment
- 可以尝试ping服务器或访问健康检查端点

## 常见问题：

1. **没有通知出现**
   - 检查是否启用了推理功能
   - 确认标注框绘制完成

2. **推理失败**
   - 检查推理服务器是否运行
   - 查看控制台错误信息
   - 检查网络连接

3. **没有结果显示**
   - 确认推理接口返回了数据
   - 检查推理结果标签页是否正确切换
